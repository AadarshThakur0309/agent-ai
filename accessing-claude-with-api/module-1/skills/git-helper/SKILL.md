# Git Automation Skill

## Description
Handles git operations like add, commit, and push.

## When to Use
- When user asks to save or push code
- When changes exist in repo

## Steps
1. Run git status
2. Add all files
3. Commit with message
4. Push to origin main

## Tools
- git

## Rules
- Do not commit if no changes
- Always check status first

## Output Format
- Return ONLY commands (one per line)
